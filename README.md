# NFT Website

A modern NFT marketplace website built with HTML and Tailwind CSS.

## Features

- Responsive design
- Modern UI with Tailwind CSS
- NFT collections showcase
- Creator profiles
- Clean and minimal design

## Getting Started

### Option 1: Direct Opening
Simply open `index.html` in your browser.

### Option 2: Development Server
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser

## Project Structure

```
nft-website/
├── index.html          # Main HTML file
├── img/               # Image assets
├── package.json       # Project configuration
└── README.md         # This file
```

## Customization

- Colors can be modified in the Tailwind config within `index.html`
- Images are located in the `img/` folder
- All styling is done with Tailwind CSS classes

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Responsive design principles
